4个概念  
1. CLI 命令行接口  
我们通过文本命令和计算机交互  
2. Terminal 终端  
在终端输入文本命令  
如iTerm，Warp  
3. Shell 命令解释器  
运行在终端的程序，接受命令，解释并执行。  
如bash，zsh  
4. 命令提示符 ( prompt )
由shell提供，并在终端显示。  
如我们通常见到的当前路径和Git信息  
我们可以安装命令提示符的第三方插件来自定义；也可以安装如oh-my-zsh这样的框架，其中包含命令提示符的定义  
  
之后讲了作者的环境，终端warp，shell zsh，第三方框架oh-my-zsh。  
其中他使用的字体是meslolgm nerdfont，有更多的符号支持。
